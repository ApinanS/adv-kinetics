Differential Equations

How do I write an ODE, set its initial value, and integrate it?

Three things every ODE needs

Derivative · initial value · integration range

1 · Derivative

$\dfrac{dY}{dt}\rightarrow$ Y'
$\dfrac{d^{2}Y}{dt^{2}}\rightarrow$ Y''

2 · Initial value

Y#0 sets $Y(0)$.
A 2nd-order ODE needs Y#0 and Y'#0.

3 · Integrate

integral t[0,50]
add step … by … to set the step & method

Integration method — default RKF (Runge–Kutta–Fehlberg). Also RKV (variable step) and EUL (Euler). Default step = 1/100 of the range.

Worked example · a single ODE

Integrate $\dfrac{dc}{dt}=5c^{2}$ from $c(0)=0.1$

Source Text
c' = 5*c^2
c#0.1
integral t[0,1]
trend c step 0.1
output c
trend plots the variable as it is integrated — instant visual feedback.
This ODE is nonlinear: its exact solution $c=\dfrac{c_0}{1-5c_0t}$ blows up at $t=1/(5c_0)$. Watch it on the next slide.

Interactive — drag the sliders

Nonlinear ODE  $\dfrac{dc}{dt}=k\,c^{2}$  runs away

Blow-up time $t^{*}=\dfrac{1}{k\,c_0}$
$c(t)$ over $t\in[0,1]$ — the curve steepens as the singularity approaches

Getting the numbers out

From trend table to Excel columns

  • Copy the result table from the EQUATRAN output window
  • Paste into Excel — values land in one text column
  • Select the column, then Data › Text to Columns
  • Choose Delimited → Space to split $t$ and $c$
  • Now plot or analyse the split columns

Define once · call anywhere

User-defined functions

Semicolon separates inputs from outputs. Call with name(input,output).

Sample Problems · Fixed-Bed Reactor

Can I simulate, size, stress-test, and fit a real reactor?

4.1 Reactor simulation · fixed-bed reactor

The reacting system & mole balances

$$\mathrm{A+B \rightleftharpoons 2C}\qquad \mathrm{2A+C\rightarrow D}$$

$r_1=k_1C_AC_B-k_2C_C^{2}$,  $r_2=k_3C_A^{2}C_C$
$k_1{=}10,\;k_2{=}5,\;k_3{=}2$;  $v{=}400$
$F_{A0}{=}600,\;F_{B0}{=}200$;  catalyst $W{=}1000$

Balance on catalyst weight: $\dfrac{dF_i}{dW}=\sum \nu_{ij}\,r_j$
$\dfrac{dF_A}{dW}=-r_1-2r_2$
$\dfrac{dF_B}{dW}=-r_1$
$\dfrac{dF_C}{dW}=2r_1-r_2$
$\dfrac{dF_D}{dW}=r_2$

4.1 Reactor simulation · the model file

How the balances become EQUATRAN code

Mass balance & rates
//Mass balance
FA'=-r1-2*r2;  FA#FA0
FB'=-r1;       FB#FB0
FC'=2*r1-r2;   FC#FC0
FD'=r2;        FD#FD0
//rate reaction
r1=k1*CA*CB-k2*CC^2
r2=k3*CA^2*CC
k1=10  k2=5  k3=2
Concentrations & run
//Concentration
CA=FA/v  CB=FB/v
CC=FC/v  CD=FD/v
v=400
//Initial
FA0=600  FB0=200
FC0=0    FD0=0
//Result
XA=(FA0-FA)/FA0*100
integral W[0,1000]
trend FA,FB,FC,FD step 100
output XA

Key Takeaways

1

Write the model, not the solver: type equations, mark initial values with #, and let EQUATRAN return the numbers.

2

ODEs need three things — a derivative (Y'), an initial value (Y#0), and an integral range; default method is RKF.

3

Wrap a reactor model in a function to size (find W), sweep sensitivity (repeat T0), and fit parameters (find … least).

4

Numerical accuracy is a modelling choice — step size changed the sizing answer from 5768 to 6350.

Write one equation system from your own project and model it in EQUATRAN before next class.
